# YuQ-Mirai

YuQ-Mirai 是一个依赖于 Mirai 的 YuQ Framework 的机器人实现。

YuQ Framework 的目的就是让机器人开发变得更简单和更有效率。

这是一个 Demo 项目。  
框架依旧处于开发阶段，我将尽可能的保证 API 尽量少的变动。

## 使用方法：
clone

run wiki.IceCream.yuq.demo.Start
项目打成jar包: mvn package
